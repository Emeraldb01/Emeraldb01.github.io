Timeline: Scientific Events Browser

Web Application for Event Visualization and User Contributions


Year: 2025 | Team Project | Role: Product Manager


Timeline is a web platform that visualizes historical scientific events in an interactive timeline. It allows users to filter events by category or year, contribute new entries, and view trending topics through a contributor leaderboard system.



The platform focuses on accessible data navigation and gamified engagement, with features like admin-reviewed event submissions, color-coded categories, and dynamic filtering options.

Key Features:
- Interactive timeline view with zoom and filters
- Contributor leaderboard and event approval system
- Year and category-based event exploration
- Admin dashboard for category management
- Automated ranking of trending categories

Results

The project demonstrated successful coordination of frontend, backend, and database integration under Agile sprints. All major milestones — including admin dashboards, user profiles, CRUD functionalities, and timeline visualization — were delivered on schedule.

Through this project, I practiced cross-functional communication, milestone planning, backlog grooming, and balancing scope against team bandwidth for an MVP delivery under tight academic deadlines.
