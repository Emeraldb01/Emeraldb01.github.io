Sign Input

Android-Based Input Method for Deaf Individuals

2nd Place in Meichu Hackathon
Year: 2023 | Team Project | Role: Coordinator, UI Engineer


Sign Input addresses a unique challenge faced by deaf individuals: typing in Chinese, a language heavily reliant on sound, in a timely and efficient manner. Many deaf individuals struggle to learn the phonetic components of Chinese characters, making traditional input methods difficult to use.



The app provides an Android-based input method that offers word-by-word translation, grammar correction, and emotion detection for accurate punctuation. This approach simplifies the typing process and ensures clarity in communication, making it a valuable tool for accessibility and inclusivity.

Key Features:
- Word-by-word translation tailored for Chinese input.
- Grammar correction to ensure language accuracy.
- Emotion detection for automatic punctuation adjustments.
- Optimized for seamless and intuitive user interaction.

Results

The app significantly improves typing efficiency and accuracy for deaf users, enabling them to communicate more effectively. Its word-by-word translation and grammar correction features were highly praised during testing, and emotion-based punctuation added an intuitive layer to the user experience.

Future plans include expanding support for additional languages, refining emotion detection algorithms, and further enhancing the app's responsiveness for diverse user scenarios.
