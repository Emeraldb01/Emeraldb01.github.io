PROJECT

540 Gesture
Recognition

This project focuses on recognizing hand gestures as input for interactive systems. Using machine learning models, it processes real-time images of hand movements. It enables seamless control of devices through gestures, enhancing user experience.

Sign Input

Sign Input is an Android app that translates sign language into text using AI. It employs deep learning models to analyze and interpret hand gestures. The app helps bridge communication gaps for the hearing impaired by providing real-time translations.

Valorant Player Analysis

This project involves analyzing player performance data from Valorant matches. It applies statistical methods and machine learning to assess player strengths and weaknesses. The analysis helps gamers optimize their playstyles and improve their gameplay.

NOTICE Note-taking App

NOTICE is a web-based note-taking app that incorporates AI for better organization. It suggests tags and categorizes notes automatically, enhancing productivity. The app also uses natural language processing to help users quickly find relevant information from their notes.
