PROJECT

NOTICE Note-taking App

NOTICE is a web application designed to tackle a common problem for university students: missing key points when dozing off in class. It provides real-time course transcription and, when needed, compares the transcript with existing notes to generate concise, bullet-point summaries.

Sign Input

Sign Input addresses the challenge faced by deaf individuals in typing Chinese, a language that relies heavily on sound. Our Android input method offers a word-by-word translation, grammar correction, and emotion detection for accurate punctuation, making communication more efficient and accessible.

Valorant Player Analysis

Our project explores the impact of temporal context and agent selection on player emotions and behavior in Valorant, a team-based multiplayer shooter. Focusing on online community dynamics, we analyze how agent race correlates with toxicity in r/Valorant comments, uncovering patterns of bias and their influence on community interactions.

540 Gesture
Recognition

The 540 Gesture Recognition project focuses on detecting 540 emergency gestures within long-form media. It combines individual gesture recognition with algorithms to identify continuous gestures, ensuring accurate detection of critical actions in real-time scenarios.
