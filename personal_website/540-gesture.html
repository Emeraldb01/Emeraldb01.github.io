540 Gesture Recognition

Using Feedforward Neural Network & Mediapipe

The goal of this project is to detect the existence of the 5-4-0 emergency gesture in a ten minutes long video.
We first used Mediapipe framework to converse hand image into landmark information.
Then the significant landmarks are trained on feedforward network, since our input size is static.
The model classified the landmark as 5, 0, 4, or others, we then uses an algorithm to detect the continuous appearance of the three gestures.
